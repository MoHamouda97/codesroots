برامج محاسبية وإدارية (Desktop Applications)

برامج الديسك توب التى تقدمها الشركة هى مجموعة مختلفة من النظم يتم وضعها بواسطة فريق العمل الخاص بالشركة بما يتناسب مع أعمال المشروع أو الشركة أو الهيئة أو الموسسة التى يتم وضع النظم اﻷلكترونية لها اعتمادا على وسائل التكنولوجيا الحديثة والتى تهدف الى تقليل الوقت والجهد والتكلفة التى تلزم ﻻنجاز اﻻعمال داخل المنظمة التى يتم التعامل معها .
